name: Design is critical at all times
description: Every piece of UI output must meet luxury design standards — no shortcuts
type: feedback

Design quality is non-negotiable on this platform. Every section, card, nav, and component must look and feel premium at all times.

**Why:** This is a luxury wedding directory positioning itself as "the intelligence layer for luxury wedding destinations." The design IS the product.

**How to apply:** Before shipping any UI change — ask: does this match the quality bar set by GCard, RegionHero, the existing dark-mode palette? If not, redesign it. Never ship placeholder-looking UI, text-only cards where images belong, or unstyled fallback states.
